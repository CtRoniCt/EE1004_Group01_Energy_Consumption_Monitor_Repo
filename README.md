__Step 1:__ change the package name from e_save to your own designated package.\
__Step 2:__ for showing the clear code outputs it is needed to open the java files from the main java class. If the desired outcome is to see the Gui interface then follow Step 3:\
__Step 3:__ to view the Gui interface it is needed to open run the EnergyMonitorGUI.java file.\
Explanation for Step 2: after running the main.java file there will be a output monitor which will ask you some questions for example, what is your name etc etc. Follow those instructions by typing the data required. after finishing all the parts a summary report will appear.\
Explanation for Step 3: Fill the blank spaces on the left hand side of the interface, select the appliance type and then verify if the entered values are correct by pressing the "Add Appliance" button, if everything is written as it is implied then you will be able to see on the right hand side, the appliance added if it doesnt show up that means an error has happend, for that a message written in red will apear and tell the exact problem that occured. Finally to get the report press the button named Generate Report, showing up a pop up screen of the report.
